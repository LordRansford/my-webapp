# Images for About Page

This directory contains professional images and certification badges for the About page.

## Required Images

1. **ransford-amponsah.jpg** - Professional headshot of Ransford Chung Amponsah
   - Recommended size: 512x512px or higher
   - Format: JPEG optimized for web
   - Should be a high-quality professional photo

2. **imeche-logo.jpg** - Institution of Mechanical Engineers logo/membership card
   - Should show Council Member status if possible
   - Format: JPEG or PNG
   - Transparent background preferred for PNG

3. **togaf-badge.png** - TOGAF Certification badge
   - Official badge from The Open Group or Credly
   - Format: PNG with transparent background preferred
   - Can be downloaded from: https://www.credly.com/badges/d36678f6-a316-46d9-8242-6b673d3b853e/public_url

## Image Optimization

All images should be:
- Optimized for web (compressed without losing quality)
- Properly sized (not excessively large)
- In modern formats (WebP versions can be added for better performance)
- Accessible (appropriate alt text is already included in the code)

## Adding Images

To add the images:
1. Place the files in this directory with the exact names listed above
2. Ensure they meet the quality standards
3. Test the About page to verify they display correctly
4. Consider running them through an image optimizer like ImageOptim or Squoosh
