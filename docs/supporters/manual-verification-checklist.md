# Supporter tiers manual verification checklist

## Free user (signed in)

- Can read all course content
- Can use quizzes
- Can use core tools and dashboards
- Can track progress and CPD
- Sees locked features explained calmly, without blocking learning

## Supporter

- Sees Supporter badge on account page and dashboard header
- Can access advanced dashboards
- Sees higher usage limits applied (where relevant)
- Does not lose any free learning features

## Professional

- Sees Professional badge on account page and dashboard header
- Can download templates (when downloads are enabled)
- Locked features explain what they do and how to unlock

## Server enforcement and safety

- Calling a restricted endpoint directly returns 401 or 403 with no sensitive data
- Access denials are logged without user content or tokens
- Entitlement grants are logged without sensitive data
