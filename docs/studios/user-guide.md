# Unified Studios User Guide

## Introduction

Welcome to the Unified Studios platform! This guide will help you get started with all our studios.

## Quick Start

1. **Visit the Hub**: Go to `/studios/hub` to see all available studios
2. **Select Your Role**: Choose Enterprise, Professional, Student, or Child mode
3. **Choose a Studio**: Pick the studio that matches your needs
4. **Learn First**: Start in the learning studio to understand concepts
5. **Build Real Projects**: Move to the live studio when ready

## Compute Credits

### Understanding Credits

Credits are used for compute-intensive operations. Each operation has an estimated credit cost shown before execution.

### Free Tier

- 60 seconds of compute per day (6,000 credits)
- Perfect for learning and small projects

### Purchasing Credits

1. Go to `/compute` to view your balance
2. Select a credit package
3. Complete payment via Stripe
4. Credits are added immediately

### Credit Packages

- **Starter**: $5 for 10,000 credits
- **Professional**: $20 for 50,000 credits (best value)
- **Enterprise**: $100 for 300,000 credits

## Child Mode

Child mode provides a safe, simplified interface for young learners:

- Large, colorful buttons
- Visual learning aids
- Pre-approved templates only
- No real credentials or data
- Game-based learning

Access child mode from any studio's settings or directly at `/studio-studio/children-mode`.

## Troubleshooting

### Credits Running Low?
- Check your usage at `/compute`
- Purchase more credits if needed
- Optimize operations to use fewer credits

### Can't Find a Feature?
- Check your audience profile (Enterprise has more features)
- Use the search function
- Check the help documentation

### Something Not Working?
- Check the browser console for errors
- Try refreshing the page
- Clear browser cache if needed
- Contact support if issues persist
