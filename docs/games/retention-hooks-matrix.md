# Retention Hooks Matrix

**Version:** 1.0  
**Date:** Current  
**Status:** Design Complete  
**Target Platform:** Browser (Next.js), Mobile-First

---

## Executive Summary

This document maps all retention hooks across all 5 new games, providing a comprehensive view of how each game maintains long-term engagement. Retention hooks are mechanisms that bring players back repeatedly and maintain their interest over time.

---

## Retention Hook Categories

### 1. Daily Challenges
- **Definition**: Unique challenges generated daily for all players
- **Purpose**: Create daily return habit, shared community experience
- **Implementation**: Seeded challenges based on date

### 2. Streaks
- **Definition**: Consecutive days of engagement
- **Purpose**: Habit formation, loss aversion, visual progress
- **Implementation**: Track consecutive days, streak freeze (1/month)

### 3. Social Sharing
- **Definition**: Share results, solutions, achievements with others
- **Purpose**: Social currency, friendly competition, community
- **Implementation**: Challenge codes, emoji grids, achievement badges

### 4. Variable Rewards
- **Definition**: Unpredictable unlocks, discoveries, achievements
- **Purpose**: Dopamine-driven engagement, surprise element
- **Implementation**: Hidden achievements, rare unlocks, unexpected rewards

### 5. Investment
- **Definition**: Player-created content, saved progress, accumulated value
- **Purpose**: Sunk cost effect, personal attachment, long-term value
- **Implementation**: Saved solutions, templates, progress tracking

### 6. Aspirational Goals
- **Definition**: Difficult challenges, rare achievements, expert status
- **Purpose**: Long-term goals, status achievement, mastery
- **Implementation**: Expert challenges, rare achievements, mastery tiers

## Detailed Retention Hooks by Game

### Constraint Optimizer

#### 1. Daily Efficiency Challenges
- **Hook**: Daily challenge code `COPT-YYYY-MM-DD-XXXX`
- **Retention Mechanism**: Compare efficiency scores with others
- **Frequency**: Daily
- **Engagement Type**: Social comparison, friendly competition
- **Why It Works**: Daily return creates habit, comparison creates motivation

#### 2. Efficiency Streaks
- **Hook**: Consecutive days with >80% efficiency scores
- **Retention Mechanism**: Streak freeze (1/month), streak milestones (7, 30, 100 days)
- **Frequency**: Daily
- **Engagement Type**: Habit formation, loss aversion
- **Why It Works**: Streak protection reduces anxiety, milestones provide goals

#### 3. Solution Gallery
- **Hook**: View how others solved (after completing)
- **Retention Mechanism**: Learn from others, favorite clever solutions
- **Frequency**: After each daily challenge
- **Engagement Type**: Social learning, investment
- **Why It Works**: Learning creates value, favoriting creates investment

#### 4. Constraint Mastery Badges
- **Hook**: Unexpected unlocks, rare badges (1% of players)
- **Retention Mechanism**: Variable reward, status symbol
- **Frequency**: Unpredictable
- **Engagement Type**: Variable reward, discovery
- **Why It Works**: Unexpected rewards create excitement, rare badges provide status

#### 5. Solution Favorites
- **Hook**: Save clever solutions for reference
- **Retention Mechanism**: Personal collection accumulates value
- **Frequency**: Ongoing
- **Engagement Type**: Investment, personal library
- **Why It Works**: Accumulated favorites create attachment, reference value

#### 6. Weekly "Impossible" Challenges
- **Hook**: Maximum complexity challenges (5% solve rate)
- **Retention Mechanism**: Aspirational goals, expert status
- **Frequency**: Weekly
- **Engagement Type**: Aspirational, mastery
- **Why It Works**: Expert challenges provide long-term goals, rare completion feels rewarding

### Memory Palace

#### 1. Spaced Repetition Algorithm
- **Hook**: Automatic review scheduling based on memory strength
- **Retention Mechanism**: Algorithm brings you back when memories need review
- **Frequency**: Algorithm-determined (1 day, 3 days, 7 days, etc.)
- **Engagement Type**: Automatic return, scientific backing
- **Why It Works**: Optimal spacing maximizes retention, algorithm creates automatic return

#### 2. Daily Memory Challenges
- **Hook**: New content to memorize each day

- **Engagement Type**: Habit formation, content variety

## Retention Hook Effectiveness Analysis

### High-Impact Hooks (Strongest Retention)

1. **Spaced Repetition (Memory Palace)**: Automatic return mechanism
2. **Daily Challenges (All Games)**: Daily habit formation
3. **Streaks (All Games)**: Loss aversion, visual progress
4. **Investment Hooks (Solutions, Templates, Palace)**: Personal attachment
5. **Social Sharing (All Games)**: Social currency, community

### Medium-Impact Hooks (Good Retention)

1. **Variable Rewards (Badges, Discoveries)**: Surprise element
2. **Progression (Techniques, Tiers)**: Clear goals
3. **Aspirational Goals (Expert Challenges)**: Long-term targets
4. **Learning Tools (Galleries, Sharing)**: Educational value

### Frequency vs. Impact

**Daily Hooks** (Highest Frequency):
- Daily challenges
- Streaks
- Speed challenges

**Weekly Hooks** (Medium Frequency):
- Expert challenges
- Complex challenges

**Unpredictable Hooks** (Variable Frequency):
- Achievement unlocks
- Pattern discoveries
- Bottleneck discoveries

**Ongoing Hooks** (Continuous):
- Investment accumulation
- Progress tracking
- Technique mastery
