# Cybersecurity Notes  
## Practitioner track  
### Role based decision making

<Callout variant="concept">
This track is not about knowing terms. It is about making decisions that affect real people. Each page is a role. Each section is a decision. Each decision is tied to evidence and consequences.
</Callout>
